Question 1.

Answer > DOM stands for Document Object Model. It is a programming interface for web documents.
The purpose of DOM is to provide a way for programs and scripts to dynamically access and update the content, structure, and style of a web page.

Question 2.

Answer > You can select HTML elements using DOM by using various methods. Here are some of the most commonly used DOM methods for selecting HTML elements:
1. getElementById(): This method selects an element based on its id attribute.
2. getElementsByClassName(): This method selects elements based on their class name.
3. getElementsByTagName(): This method selects elements based on their tag name.
4. querySelector(): This method selects the first element that matches a CSS selector.
5. querySelectorAll(): This method selects all elements that match a CSS selector.

Answer > Event bubbling is a mechanism in which events are first captured by the innermost element and then propagated or bubbled up through its parent elements, all the way to the outermost element. During this process, each element can perform its own event handling.
